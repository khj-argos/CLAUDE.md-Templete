# 설명
지금까지 온갖 삽질 경험을 통해 만든 CLAUDE.md를 공개합니다.

Claude Code에 이 내용들만 넣어도 성능 훨씬 좋아질거에요.

이거 쓰실 때 직접 CLAUDE.md 코드 건들지 마시고요, 그냥 Claude Code에 때려 넣으시고 

"이 중 내 프로젝트에 적용할만한 것들만 우리 CLAUDE.md에 반영해줘" 라고 하면 

알아서 CLAUDE.md에 적용해 줄거에요.

# 사용 방법
1. 프로젝트에 맞는 템플릿 선택
2. 복사하기 버튼 클릭
3. 프로젝트 루트에 CLAUDE.md 파일 생성
4. 복사한 내용 붙여넣기
5. 프로젝트에 맞게 커스터마이징



# 사용 팁
• 여러 템플릿 조합: Next.js + Firebase 프로젝트라면 두 템플릿을 합쳐서 사용하세요

• 프로젝트별 커스터마이징: 템플릿은 시작점일 뿐, 프로젝트 특성에 맞게 수정하세요

• 팀과 공유: CLAUDE.md는 팀 전체가 Claude Code를 효과적으로 사용하게 도와줍니다

• 지속적 업데이트: 프로젝트가 발전하면서 CLAUDE.md도 함께 업데이트하세요
